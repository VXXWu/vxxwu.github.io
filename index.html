ABOUT THE INTERACTIVE MAP

Geographic information is difficult to intuitively understand, but by utilizing visualization tools like interactive maps, geographic data can be examined in an easily processable format to develop new research questions and see relationships.

This interactive map can display a large dataset of 30 different basic demographics listed below and can additionally display part of a professionally gathered dataset on crowdfunding. Enjoy!
